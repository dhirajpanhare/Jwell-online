================================================================================
  TROUBLESHOOTING
================================================================================

Issue: Foreign key constraint error
Solution: Ensure parent records exist before inserting child records

Issue: Slow product search
Solution: Rebuild fulltext index - REPAIR TABLE Products

Issue: Out of disk space
Solution: Delete old execution logs - DELETE FROM ExecutionLogs WHERE CreatedAt < DATE_SUB(NOW(), INTERVAL 90 DAY)

Issue: Duplicate email error
Solution: Email already exists - check Users table with: SELECT * FROM Users WHERE Email = 'email@example.com'

Issue: Stock goes negative
Solution: Implement stock validation in SP_AddCartItem and SP_CreateOrder
